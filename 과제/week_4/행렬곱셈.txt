#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>

main() {
	int A[102][102], B[102][102];
	int N, M, K;

	scanf("%d %d", &N, &M);
	for (int i = 0; i < N; i++) {
		for (int j = 0; j < M; j++) {
			scanf("%d", &A[i][j]);
		}
	}

	scanf("%d %d", &M, &K);
	for (int i = 0; i < M; i++) {
		for (int j = 0; j < K; j++) {
			scanf("%d", &B[i][j]);
		}
	}

	for (int i = 0; i < N; i++) {
		for (int j = 0; j < K; j++) {
			int sum = 0;
			for (int p = 0; p < M; p++) {
				sum += A[i][p] * B[p][j];
			}
			printf("%d",sum);
		}
		printf("\n");
	}
}



이 코드는 입력으로 받은 두 개의 행렬 A와 B를 곱하는 C언어 프로그램입니다.

먼저 _CRT_SECURE_NO_WARNINGS를 정의하여 scanf() 함수를 안전하게 사용합니다.

N, M, K를 입력받습니다.

N개의 줄에 대하여, M개의 정수를 입력받아 행렬 A를 구성합니다.

M, K를 입력받습니다.

M개의 줄에 대하여, K개의 정수를 입력받아 행렬 B를 구성합니다.

행렬 A와 B를 곱하여 결과 행렬을 출력합니다.

6-1. 행렬 A의 i행과 행렬 B의 j열을 곱하는 연산을 수행합니다.

6-2. 곱한 값을 sum 변수에 누적합니다.

6-3. 모든 p에 대하여 A[i][p]와 B[p][j]의 곱을 계산합니다.

6-4. sum 변수에 누적된 값을 출력합니다.

6-5. 한 행의 곱셈이 끝날 때마다 줄바꿈 문자를 출력하여 결과를 N개의 줄로 구분합니다.





해당 코드는 C언어로 작성된 행렬 곱셈 프로그램입니다.

프로그램은 총 3개의 배열과 3개의 변수를 사용합니다. 첫 번째 배열인 A는 NM 크기의 행렬을 저장하는 배열이고, 두 번째 배열인 B는 MK 크기의 행렬을 저장하는 배열입니다. 결과값을 저장하는 배열은 따로 없으며, A와 B 행렬을 곱한 결과를 바로 출력하게 됩니다.

N, M, K 변수는 각각 A 행렬의 행의 개수, A 행렬의 열의 개수 (동시에 B 행렬의 행의 개수), 그리고 B 행렬의 열의 개수를 나타냅니다.

다음은 코드에서 이루어지는 과정입니다.

N, M, K를 입력받습니다. (첫 번째 줄)
A 행렬을 입력받습니다. (두 번째 줄부터 N개의 줄)
B 행렬을 입력받습니다. (N+2번째 줄부터 M개의 줄)
A와 B를 곱하여 결과를 출력합니다.
이때, 행렬 곱셈을 하기 위해서는 A의 한 행과 B의 한 열을 곱한 결과를 더해야 합니다. 이를 위해 for문을 3번 중첩하여 사용하게 됩니다.

첫 번째 for문은 A의 각 행에 대해 반복하며, 두 번째 for문은 B의 각 열에 대해 반복합니다. 세 번째 for문은 A의 한 행과 B의 한 열을 곱한 값을 더하는 과정을 반복합니다. 이때, 더한 값은 sum 변수에 저장됩니다.

각 행렬의 원소는 정수형으로 주어지므로, sum 변수도 정수형으로 선언됩니다. 행렬 곱셈 결과가 정수이기 때문입니다.

모든 반복문이 끝나면, 출력문을 통해 A와 B의 곱셈 결과를 출력합니다. 출력시 곱셈 결과를 공백으로 구분하여 출력하며, 한 행이 끝나면 다음 행으로 넘어가기 위해 개행 문자를 출력합니다.
